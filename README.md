# 诗歌朗诵

一个简单的诗歌朗诵网站。

网页地址：https://poetry-reader.heyfe.org/

## 介绍

-   诗歌数据通过 [今日诗词](https://www.jinrishici.com/doc/) `API` 获取
-   通过 `Web Speech API` 进行诗歌朗诵
-   使用 `Web Animations API` 做了简单动画
